# dw--aula04

Este projeto contém exemplos práticos de conceitos fundamentais da linguagem JavaScript, demonstrados através do arquivo `tf-04.js`.

## Conceitos Demonstrados

### 1. Funções Básicas
- **Função `somar(a, b)`**: Realiza a soma de dois números.
- **Função `cumprimentar(nome)`**: Retorna uma saudação personalizada.

### 2. Objetos e Métodos
- **Objeto `calculadora`**: Contém métodos para operações matemáticas básicas como soma e subtração.

### 3. Callbacks
- **Função `executarOperacao(a, b, operacao)`**: Demonstra o uso de callbacks para executar operações de forma dinâmica.

### 4. Operações Assíncronas
- **setTimeout**: Exemplo de execução de código após um atraso de tempo.
- **Promises**: Demonstração de operações assíncronas com tratamento de sucesso e erro usando `.then()` e `.catch()`.

## Como Executar
Para executar o código, use Node.js:
```
node tf-04.js
```

O código irá imprimir os resultados das operações no console, incluindo saídas de funções síncronas e assíncronas.
